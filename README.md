# PDF Combiner

A fully client-side PDF merging tool. I didn't feel comfortable uploading personal documents to use one of those online tools so just decided to quickly build my own version. All processing happens in the browser & no files are ever uploaded to a server.

## Features

- Drag and drop or browse to select PDF files
- Reorder files
- Remove individual files or clear all
- Displays file names, sizes, and total combined size

## Tech Stack

- React + TypeScript
- Vite
- pdf-lib

## Getting Started

```bash
npm install
npm run dev
```
